Perl Code Guideline
*********************************
1. SubRoutine Call
Always include "&" before the subroutine and "()" after the subroutine
&Package::SubRoutine(); # Call subroutine in another named package
&::SubRoutine();# Call subroutine from main package
&SubRoutine(); # Inside same package

2. SubRoutine Declaration
sub SubRoutine(@)
{
}
sub SubRoutine(\@)
{
}
sub SubRoutine($)
{
}
